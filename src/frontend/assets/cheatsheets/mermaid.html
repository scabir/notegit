Mermaid Diagrams Cheatsheet

Wrap Mermaid diagrams in a ```mermaid code block.

Some diagrams are marked as beta and may require newer Mermaid versions.

Flowchart

```mermaid
flowchart LR
  A[Start] --> B{Decision}
  B -->|Yes| C[Continue]
  B -->|No| D[Stop]
  subgraph Phase 1
    C --> E[Finish]
  end
```

Sequence Diagram

```mermaid
sequenceDiagram
  participant U as User
  participant A as App
  participant S as Service
  U->>A: Click
  A->>S: Request
  S-->>A: Response
  A-->>U: Update
```

Class Diagram

```mermaid
classDiagram
  class User {
    +string name
  }
  class Note {
    +string title
    +string body
  }
  User "1" --> "*" Note : owns
```

State Diagram (v2)

```mermaid
stateDiagram-v2
  [*] --> Idle
  Idle --> Editing : open
  Editing --> Saved : save
  Saved --> Idle : close
```

ER Diagram

```mermaid
erDiagram
  CUSTOMER }|..|{ ORDER : places
  ORDER ||--|{ ORDER-ITEM : includes
```

Gantt

```mermaid
gantt
  title Project Plan
  dateFormat  YYYY-MM-DD
  section Build
  Setup :a1, 2025-01-01, 3d
  Dev   :a2, after a1, 5d
```

User Journey

```mermaid
journey
  title User Journey
  section Search
    Find note: 5: User
    Open note: 4: User
```

Pie

```mermaid
pie title Time
  "Coding" : 8
  "Meetings" : 2
```

Requirement Diagram

```mermaid
requirementDiagram
  requirement req1 {
    id: REQ-1
    text: Save notes
    risk: medium
    verifymethod: test
  }
  element app
  app - satisfies -> req1
```

Git Graph

```mermaid
gitGraph
  commit
  branch feature
  checkout feature

  checkout main
  merge feature
```

Quadrant Chart

```mermaid
quadrantChart
  title Priorities
  x-axis Low --> High
  y-axis Low --> High
  quadrant-1 Urgent
  quadrant-2 Important
  quadrant-3 Defer
  quadrant-4 Optional
  "Fix bug" : [0.8, 0.9]
```

