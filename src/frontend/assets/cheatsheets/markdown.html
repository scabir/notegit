Markdown Format Cheatsheet

Quick reference for common Markdown syntax.

Headings

# H1
## H2
### H3

Emphasis

**bold**
*italic*
~~strikethrough~~

Lists

- bullet item
1. numbered item

Task List

- [ ] Task
- [x] Done

Links and Images

[link text](https://example.com)
![alt text](./path/to/image.png)

Code

`inline code`

```language
code block
```

Table

|  | Col1 | Col2 |
| --- | --- | --- |
| Row1 |  |  |
| Row2 |  |  |

Footnote

Text with a footnote[^1]

[^1]: Footnote text

Highlight

I need to highlight these ==very important words==.

Definition List

term
: definition

Mermaid

```mermaid
flowchart TB
  A --> B
```
