# How to Create a Repo on NoteBranch

This tutorial was generated automatically with Playwright using the local NoteBranch app in mock Git mode.

## Step 1: Open NoteBranch and start setup

When you open NoteBranch for the first time, use the 'Connect to Repository' button to start creating your repo connection.

![Open NoteBranch and start setup](images/step-01-welcome-screen.png)

## Step 2: Open the repository connection dialog

The dialog asks for your repository details. Stay on the Git tab for a standard Git remote setup.

![Open the repository connection dialog](images/step-02-open-connect-dialog.png)

## Step 3: Fill remote URL, branch, and token

Provide the repository URL, the target branch, and your personal access token, then review values before connecting.

![Fill remote URL, branch, and token](images/step-03-fill-repository-details.png)

## Step 4: Finish and verify repository is connected

After connecting, the main workspace appears. The status bar and file tree confirm the repo is ready to use.

![Finish and verify repository is connected](images/step-04-repository-connected.png)
