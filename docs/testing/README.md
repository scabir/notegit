# Testing

## Run tests

```bash
pnpm test
```

## Coverage

```bash
pnpm run test:coverage
```

Coverage reports are written to `coverage/` and include both frontend and backend unit tests.

## Watch mode

```bash
pnpm run test:watch
```

## Test layout

- `src/unit-tests/backend/`
- `src/unit-tests/frontend/`
- `src/unit-tests/setup.ts`
